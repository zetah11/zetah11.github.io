The keyword game

What is this?

You'll be presented with a list of hidden words, each of which is a keyword in the language you choose. As you type out words, those which appear on the list will be revealed. By default, the words will be placed in a random order. Check “Alphabetical order” to instead order them.

For languages where keywords are case insensitive, the game will also be case insensitive.

What exactly is a keyword? The answer depends on the language and is often quite fuzzy. In some languages, whether a word is classified as a keyword or not can depend on all kinds of context and which specific implementation you're using. For the purposes of this quiz, an “include as many as is reasonable”-approach is taken: if it looks a bit like an identifier or name, but the language gives it some special meaning or syntax, it's a keyword.
